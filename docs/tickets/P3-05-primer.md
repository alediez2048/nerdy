# P3-05 Primer: Single-Variable A/B Variants

**Project:** Ad-Ops-Autopilot  
**Phase:** 3 — Multi-Modal Ads (v2)

## Description

Control ad + 3 variants each changing exactly one element (R2-Q6): hook type, emotional angle, or CTA. Compare each variant against control. Each variant is a learning investment, not a lottery ticket.

**Acceptance:** Winning patterns identified per audience segment; causal impact of each variable measurable.
