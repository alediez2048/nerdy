# P1-05 Primer: Campaign-Goal-Adaptive Weighting

**Project:** Ad-Ops-Autopilot  
**Phase:** 1 — Core Pipeline (v1)

## Description

Awareness vs. conversion weight profiles (R1-Q3). Clarity ≥ 6.0 and Brand Voice ≥ 5.0 floor constraints. System selects profile based on campaign goal in the brief; floor violations trigger automatic rejection.

**Acceptance:** Correct weights per campaign goal; floor violations → rejection.
