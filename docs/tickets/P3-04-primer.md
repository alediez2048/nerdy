# P3-04 Primer: Text-Image Coherence Verification

**Project:** Ad-Ops-Autopilot  
**Phase:** 3 — Multi-Modal Ads (v2)

## Description

Post-generation coherence check (R1-Q10). "Does this image reinforce the message and emotion of the copy?" Mismatched pairs flagged for image regeneration only (cheaper than regenerating both).

**Acceptance:** Mismatched pairs flagged; image regeneration triggered when coherence below threshold.
