# P3-01 Primer: Shared Semantic Brief Expansion

**Project:** Ad-Ops-Autopilot  
**Phase:** 3 — Multi-Modal Ads (v2)

## Description

Expand brief into visual spec (R1-Q10): tone, setting, demographic, color palette, key object/action. Both text and image generators receive this shared input. Prevents multi-modal incoherence by design.

**Acceptance:** Both generators receive shared visual spec; coherence architecturally enforced through shared input.
