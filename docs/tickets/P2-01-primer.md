# P2-01 Primer: Inversion Tests

**Project:** Ad-Ops-Autopilot  
**Phase:** 2 — Testing & Validation

## Description

Degrade one dimension at a time in high-scoring ads (R2-Q3). Verify only the degraded dimension's score drops (≥1.5); others stay stable (±0.5). Proves the evaluator measures 5 independent dimensions, not a halo effect.

**Acceptance:** 10+ tests; degraded dimension drops ≥1.5, others stable ±0.5.
