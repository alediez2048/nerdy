# P1-07 Primer: Pareto-Optimal Regeneration

**Project:** Ad-Ops-Autopilot  
**Phase:** 1 — Core Pipeline (v1)

## Description

Generate 3–5 variants per regeneration cycle; select the variant that is Pareto-dominant (R1-Q5). No other variant scores higher on all dimensions simultaneously. Prevents dimension collapse.

**Acceptance:** No dimension regressed vs. prior cycle; Pareto selection structurally prevents trade-offs.
