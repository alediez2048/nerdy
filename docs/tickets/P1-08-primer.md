# P1-08 Primer: Brief Mutation + Escalation

**Project:** Ad-Ops-Autopilot  
**Phase:** 1 — Core Pipeline (v1)

## Description

After 2 failed regeneration attempts: mutate the brief based on the persistently weak dimension (R1-Q2). After 3 failures: escalate to human review with full diagnostic package. Fix root cause before giving up.

**Acceptance:** Mutation logged; escalation triggers on third failure; diagnostic package includes scores, prompts, and weak dimension analysis.
