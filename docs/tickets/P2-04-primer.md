# P2-04 Primer: SPC Drift Detection

**Project:** Ad-Ops-Autopilot  
**Phase:** 2 — Testing & Validation

## Description

Statistical process control on evaluator score distributions (R1-Q1). Control charts with ±2σ limits from rolling 5-batch mean. When batch breaches limits, inject canary ads to diagnose drift vs. real quality change.

**Acceptance:** Control charts plotted; canary injection fires on simulated drift.
