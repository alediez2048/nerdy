# P4-03 Primer: Competitive Intelligence Pipeline

**Project:** Ad-Ops-Autopilot  
**Phase:** 4 — Autonomous Engine (v3)

## Description

Structured pattern extraction from Meta Ad Library (R2-Q2). LLM categorizes each ad: hook type, emotional angle, CTA style, target audience. Store as queryable pattern records. Generator queries this database. +10 bonus points.

**Acceptance:** Pattern database populated; generator queries it; structured, queryable patterns from raw ads.
