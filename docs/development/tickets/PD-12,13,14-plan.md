# PD-12, PD-13, PD-14 — Scoring Expansion Plan

## Overview
Expand the scoring system from copy-only (5 dimensions) to a unified 20-dimension framework across 4 categories. Each category uses 1-10 scale with per-dimension rationales via Gemini 2.0 Flash multimodal.

## Scoring Framework (20 dimensions)

### Copy Score (existing — 5 dims)
Already implemented in `evaluate/evaluator.py`. No changes needed.

| Dimension | What it measures |
|-----------|-----------------|
| Clarity | Is the message immediately understandable? |
| Value Proposition | Does the ad communicate a compelling benefit? |
| CTA | Does the ad drive the reader to act? |
| Brand Voice | Does the tone match Varsity Tutors brand? |
| Emotional Resonance | Does the ad connect emotionally with the audience? |

### PD-12: Brief Adherence Score (new — 5 dims)
Measures: "Did the ad do what I asked?"

| Dimension | What it measures |
|-----------|-----------------|
| Audience Match | Does it speak to the configured audience (parents vs students)? |
| Goal Alignment | Does it serve the campaign goal (conversion vs awareness)? |
| Persona Fit | Does the tone match the persona's emotional state? |
| Message Delivery | Is the key message present or clearly conveyed? |
| Format Adherence | Does it match the creative brief style (ugc_testimonial, etc.)? |

**Inputs:** Ad copy + session config (audience, campaign_goal, persona, key_message, creative_brief) + image/video if present
**New file:** `evaluate/brief_adherence.py`
**Ledger event:** `BriefAdherenceScored`
**Backfill:** All 146 existing ads — copy + config only, no media needed (~$0.02)

### PD-13: Image Quality Score (new — 5 dims)
Measures: "How good is the image as a social ad creative?"

| Dimension | What it measures |
|-----------|-----------------|
| Visual Clarity | Is the subject immediately identifiable? Clean composition? |
| Brand Consistency | Colors match Varsity Tutors palette? Professional tone? |
| Emotional Impact | Does the image evoke the intended emotion? |
| Copy-Image Coherence | Does the visual reinforce the ad text? |
| Platform Fit | Mobile-optimized? Readable at small size? |

**Inputs:** Image file (multimodal) + ad copy for coherence check
**New file:** `evaluate/image_scorer.py`
**Ledger event:** `ImageScored`
**Backfill:** ~58 existing images (~$0.03)

### PD-14: Video Quality Score (new — 5 dims)
Measures: "How good is the video as a social ad creative?"

| Dimension | What it measures |
|-----------|-----------------|
| Hook Strength | Do the first 2 seconds grab attention? |
| Visual Quality | Smooth motion, good lighting, no artifacts? |
| Narrative Flow | Clear beginning-middle-end? Pacing matches message? |
| Copy-Video Coherence | Does the video reinforce the ad text and CTA? |
| UGC Authenticity | Feels genuine vs corporate? Relatable? |

**Inputs:** Video file (Gemini native video understanding) + ad copy
**New file:** `evaluate/video_scorer.py`
**Ledger event:** `VideoScored`
**Backfill:** ~9 existing videos (~$0.01)

## Architecture

```
Session Config ──────────────────────────────┐
                                             │
Ad Copy ──→ [Copy Evaluator] ──→ 5 copy dims │
         │                                   ▼
         ├─→ [Brief Adherence] ──→ 5 adherence dims (PD-12)
         │
         ├─→ [Image Scorer] ──→ 5 image dims (PD-13)
         │    (+ image file)
         │
         └─→ [Video Scorer] ──→ 5 video dims (PD-14)
              (+ video file)

All scores ──→ Ledger events ──→ Dashboard panels (PD-15, future)
```

## Pipeline Integration

Each scorer is called after the existing evaluation pass:

```python
# In pipeline_task.py — after ad is generated + evaluated

# PD-12: Brief adherence (runs for all ads)
adherence = score_brief_adherence(ad_copy, config, image_path, video_path)
log_event(ledger, {"event_type": "BriefAdherenceScored", ...})

# PD-13: Image scoring (runs only if image exists)
if image_path and Path(image_path).exists():
    image_scores = score_image(image_path, ad_copy, config)
    log_event(ledger, {"event_type": "ImageScored", ...})

# PD-14: Video scoring (runs only if video exists)
if video_path and Path(video_path).exists():
    video_scores = score_video(video_path, ad_copy, config)
    log_event(ledger, {"event_type": "VideoScored", ...})
```

## Backfill Strategy

One unified script: `scripts/backfill_scores.py`
- Reads all session ledgers + session configs from DB
- For each published/selected ad:
  - Run brief adherence scorer (all ads)
  - Run image scorer (if image on disk)
  - Run video scorer (if video on disk)
- Appends new events to session ledgers
- Total cost: ~$0.06, ~15 min runtime

## Relationship to Existing Evaluation

| Component | Purpose | Changes? |
|-----------|---------|----------|
| `evaluate/evaluator.py` | Copy quality scoring (5 dims) | None |
| `evaluate/image_evaluator.py` | Binary attribute gate + coherence | None — still gates publish |
| `evaluate/video_evaluator.py` | Binary attribute gate + coherence | None — still gates publish |
| `evaluate/brief_adherence.py` | Brief adherence scoring (5 dims) | **New (PD-12)** |
| `evaluate/image_scorer.py` | Image quality scoring (5 dims) | **New (PD-13)** |
| `evaluate/video_scorer.py` | Video quality scoring (5 dims) | **New (PD-14)** |

Existing evaluators decide **if** content publishes (gate).
New scorers measure **how good** it is (analytics + insights).
