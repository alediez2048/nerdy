# P1-03 Primer: Audience-Specific Brand Voice Profiles

**Project:** Ad-Ops-Autopilot  
**Phase:** 1 — Core Pipeline (v1)

## Description

Parent-facing + Student-facing sub-profiles with few-shot examples (R1-Q6). Each profile has distinct tone, emotional drivers, and on-brand examples. Generator selects profile based on audience segment in the brief.

**Acceptance:** Correct profile selected per audience segment; evaluator scores Brand Voice against audience-specific rubric.
