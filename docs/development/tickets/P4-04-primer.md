# P4-04 Primer: Cross-Campaign Transfer

**Project:** Ad-Ops-Autopilot  
**Phase:** 4 — Autonomous Engine (v3)

## Description

Shared structural patterns + isolated content (R3-Q8). Hook types, body patterns, CTA styles transfer across campaigns; specific claims and proof points stay campaign-specific. Pattern library has campaign_scope tag: universal or campaign:{name}.

**Acceptance:** Insights transferable via campaign_scope tags; craft transfers, substance doesn't.
