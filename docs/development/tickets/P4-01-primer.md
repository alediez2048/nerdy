# P4-01 Primer: Agentic Orchestration Layer

**Project:** Ad-Ops-Autopilot  
**Phase:** 4 — Autonomous Engine (v3)

## Description

Researcher → Writer → Editor → Evaluator with error boundaries (R3-Q1). Linear pipeline with bounded contracts per agent. Failures contained; diagnostics logged. Parallelism across briefs, not across agents within an ad.

**Acceptance:** Failures contained; diagnostics logged; no cascading failures or circular dependencies.
